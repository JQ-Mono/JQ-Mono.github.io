第二题：邻域均值

  • 求矩阵中每个元素满足下列条件的个数：
    • 以本元素为中心，求某一范围内的矩阵中所有元素的个数与和
    • 求得上述所有元素的平均值，该值小于给定的一个阈值
  • 思路：
    • 暴力枚举计算可得$70$分
    • 满分需要用到二维前缀和技巧
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n, L, r, t; cin>>n>>L>>r>>t;
    vector<vector<int>> vec(n, vector<int>(n));
    for(int i = 0; i < n; ++i){
        for(int j = 0; j < n; ++j){
            cin>>vec[i][j];
        }
    }
    vector<vector<int>> sum(n, vector<int>(n));
    sum[0][0] = vec[0][0];
    for(int i = 1; i < n; ++i) sum[i][0] = sum[i - 1][0] + vec[i][0];
    for(int j = 1; j < n; ++j) sum[0][j] = sum[0][j - 1] + vec[0][j];
    for(int i = 1; i < n; ++i){
        for(int j = 1; j < n; ++j){
            sum[i][j] = sum[i - 1][j] + sum[i][j - 1] - sum[i - 1][j - 1] + vec[i][j];
        }
    }
    int ans = 0;
    for(int i = 0; i < n; ++i){
        for(int j = 0; j < n; ++j){
            int x = min(n - 1, i + r), y = min(n - 1, j + r);
            int a = max(i - r, 0), b = max(j - r, 0);
            int total = sum[x][y];
            if(b > 0) total -= sum[x][b - 1];
            if(a > 0) total -= sum[a - 1][y];
            if(a > 0 && b > 0) total += sum[a - 1][b - 1];
            int cnt = (x - a + 1) * (y - b + 1);
            if(total <= t * cnt) ans++;
        }
    }
    cout<<ans;
    return 0;
}

第三题：DHCP

  • 大模拟题，考试时冲着$20%$的测试点写的，满分代码待训练。
// 20分
#include <bits/stdc++.h>
using namespace std;
int main(){
    int N, td, tmx, tmn; cin>>N>>td>>tmx>>tmn;
    string H; cin>>H;
    vector<int> ips(N + 10, -1); // -1未分配
    map<string, int> mp;
    int n; cin>>n;
    for(int i = 0; i < n; ++i){
        int t; cin>>t;
        string a; cin>>a;
        string b; cin>>b;
        string type; cin>>type;
        int ip, limit; cin>>ip>>limit;
        if(type == "DIS" && a != "*" && b == "*"){
            int choice = -1;
            if(mp.count(a) && ips[mp[a]] > t){
                choice = mp[a];
            }
            if(choice == -1){
                for(int i = 1; i <= N; ++i){
                    if(ips[i] == -1){
                        choice = i;
                        break;
                    }
                }
            }
            if(choice == -1){
                for(int i = 1; i <= N; ++i){
                    if(ips[i] <= t){
                        choice = i;
                        break;
                    }
                }
            }
            if(choice == -1) continue;
            if(limit == 0) limit = td + t;
            else {
                if(limit - t < tmn) limit = tmn + t;
                else if(limit - t > tmx) limit = tmx + t;
            }
            ips[choice] = limit;
            mp[a] = choice;
            cout<<H<<' '<<a<<" OFR "<<choice<<' '<<limit<<endl;
        }
    }
    return 0;
}

第四题：校门外的树

  • 比较经典的动态规划，思考过程有点类似于最长上升子序列问题，只是更新答案的时候计算起来有些复杂了，需要考虑因数情况。
  • 虽然$AC$代码只有不到$40$行之少，但考试的时候我也花了一个多小时的时间来思考，中途也想放弃过，好在最终想明白了，算是算法思想的小进步吧。整题的分析过程还请耐心看下去～
  • 回顾题意，给定n个位置表示障碍物排成一条直线，在这些障碍物的中间区域种数，规则是：
    • 树不可以种在障碍物上
    • 在障碍物中选定其中两个位置，在这两个位置中间种数，且要求这两个障碍物与中间种树的位置构成一组等差数列，如$1,2,3$或$5,10,15$等，同样地树不能种在障碍物上（如果两个障碍物中间有其他障碍物）
    • 将满足了第二条规则的两个障碍物记为一个区间，最终目的是将整个障碍物的大区间划分为连续不间断的小区间，且每个小区间都满足上述规则。
  • 以样例一为例子：障碍物位置是0,2,6
    • 使得0,2区间满足：在位置1处种一棵树
    • 使得2,6区间满足：在位置4处种一棵树，或者在位置3,4,5处各种一棵树，两种情况
    • 使得0,6区间满足：在位置3处种一棵树，不能以公差为1或2地种数，因为中间有障碍物
    • 本题的答案为$3$：合并考虑前两条：$1\times2$种，仅考虑第三条：$1$种，总共$3$种情况。
  • 第一步首先预处理好1e5以内每个数的因数，因为后面需要大量使用。
  • 动态规划的思路比较清晰，用dp[i-1]表示包含到第i个障碍物的位置的答案数，而要求得dp[i]的答案，需要从第i+1个障碍物开始，向前遍历每个障碍物，并更新答案。记当前遍历到第j个障碍物，那么dp[i]需要加上dp[j]乘以第j个障碍物到第i个障碍物之间种数的情况数。
  • 一种简答的暴力实现复杂度为$O(n^3)$，枚举障碍物是第一层，向前遍历是第二层，判断之间的情况数是第三层，这样做下来可以拿到$50/60$分。
  • 到此我们还有两个问题需要解决，一个是将复杂度降为$O(n^2)$，另一个是考虑两个障碍物如果相邻的情况下怎么考虑答案。
  • 关于复杂度，如果动态规划思想用的灵活的话可以发现，前两层for循环是不可避免的，唯一能动手脚的是第三层，如果能将第三层for循环合并到第二层，那么降复杂度就大功告成了。
  • 我觉得最关键的地方是如何理解两个障碍物相邻情况下的答案，这里我给出我的处理方法：
    • 举个例子：障碍物为：0,1,5，那么到1为止，之前的答案数应该为0，因为根本种不了树。
    • 但是要注意的是，答案虽然为0，但是给后面的贡献仍然需要记为1，换句话说，障碍物1到障碍物5之间的种数情况有$2$种（公差为$1$与$2$），那么这个$2$需要乘以$1$而不是$0$。
    • 本题的一个大坑点就在此，如果没有解决这个问题，那么后面$40$分就算是$O(n^2)$复杂度也难逃$Wrong\ Answer$，毕竟$1000$个障碍物出现相邻的可能性还是不小的。
  • 继续考虑如何降低复杂度，我们维护一个集合，将第二层循环内使用过的因数存入，表示已经使用过，后面不能再使用。还是以样例为例：
    • 0,2,6三个障碍物，假设当前一层遍历到6，二层将从2开始，初始化集合为空，$2$到$6$的差值为$4$，将$4$的所有因数（$1$和$2$）加入集合，本次情况数为$2$，再乘以到2为止的答案数加到一层答案中去。注意还要再向集合插入$4$这个差值。
    • 向前遍历到0，$0$到$6$的差值为$6$，将$6$的所有因数（$1$和$2$和$3$）加入集合，不过此时只有$3$一个因数插入成功，本次情况数为$1$，再乘以到0为止的答案数（虽然保存为$0$但是手动更改为$1$）
  • 至此算法设计完成，时间复杂度为$O(n^2)$，在向前遍历的过程中同时计算了答案。
// 整理后的代码
#include <bits/stdc++.h>
using namespace std;
vector<long long> primes[100010];
void init(){
    for(int i = 1; i <= 1e5; ++i)
        for(int j = i + i; j <= 1e5; j += i)
            primes[j].push_back(i);
}
int main(){
    init();
    int n; cin>>n;
    vector<long long> vec(n);
    for(int i = 0; i < n; ++i) cin>>vec[i];
    vector<long long> dp(n);
    for(int i = 1; i < n; ++i){
        unordered_set<int> s;
        for(int j = i - 1; j >= 0; --j){
            long long cur = 0;
            for(int ev: primes[vec[i] - vec[j]]){
                if(!s.count(ev)) {
                    cur++;
                    s.insert(ev);
                }
            }
            dp[i] += (dp[j] == 0 ? 1 : dp[j]) * cur;
            dp[i] %= 1000000007;
            s.insert(vec[i] - vec[j]);
        }
    }
    cout<<dp[n - 1];
    return 0;
}

第五题：疫苗运输

  • 本次第五题令我比较惊讶，因为这次第五题的题面比较好理解，没有提出一些新的定义与概念，而且就读完题后让人第一时间想去模拟（不知道是不是因为我菜，想不到AC解）。关键是，我模拟完之后能拿到$80$分，原来CSP对暴力这么友好吗？？？
  • 题意我就不再复述了，讲讲我当时的思路吧。
  • 首先由m条路线，建立m个对应的队列，队列每个元素是一个pair<int, int>类型，如<3, 100>表示从3出发经过100个单位的时间到达下一站，到达下一站之后，队头弹出并放入队尾，表示循环。
  • 用一个变量curTime表示当前时间，之后所有线路同时运转，增加curTime直到1e6结束（在数小的情况下粗略表示经过了无限时间）。运转过程是，先找到所有路线中每条路线的当前位置抵达下一站的最小时间，然后所有路线均跑动这个最小时间，调整抵达下一站的那些队列的结构，并更新答案列表。
  • 另一个细节是，每个队列还有额外一个标记flag，表示这个队列是否被激活，激活的队列有权更新答案，否则只能等到被激活。被激活的条件如下：
    • 当前队列队头为第1站
    • 当前队列队头与另一个被激活队列的队头表示的站点一样，即换站过程
  • 总体思路如上述所言，细节可以看看代码。
  • 本来只想暴力拿个$30/40$分的，但是交上去$80$分，那我也只好欣然接受了hhhhhh
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n, m, x, a, b; cin>>n>>m;
    vector<int> ans(n + 1, INT_MAX);
    vector<bool> flag(m, false);
    vector<queue<pair<int, int>>> vec(m);
    vector<int> curleft(m);
    for(int i = 0; i < m; ++i){
        cin>>x;
        while(x--){
            cin>>a>>b;
            vec[i].push({a, b});
        }
    }
    for(int i = 0; i < m; ++i){
        if(vec[i].front().first == 1) flag[i] = true;
        curleft[i] = vec[i].front().second;
    }
    int curTime = 0;
    while(curTime < 1e6){
        int mn = *min_element(curleft.begin(), curleft.end());
        curTime += mn;
        for(int i = 0; i < m; ++i){
            curleft[i] -= mn;
            if(curleft[i] == 0){
                vec[i].push(vec[i].front());
                vec[i].pop();
                curleft[i] = vec[i].front().second;
            }
        }
        for(int i = 0; i < m; ++i){
            if(flag[i] && curleft[i] == vec[i].front().second)
                ans[vec[i].front().first] = min(ans[vec[i].front().first], curTime);
        }
        for(int i = 0; i < m; ++i){
            if(vec[i].front().first == 1) flag[i] = true;
            if(!flag[i]) continue;
            for(int j = 0; j < m; ++j){
                if(!flag[j] && curleft[i] == vec[i].front().second && curleft[j] == vec[j].front().second && vec[i].front().first == vec[j].front().first){
                    flag[j] = true;
                }
            }
        }
    }
    for(int i = 2; i < ans.size(); ++i){
        if(ans[i] != INT_MAX) cout<<ans[i]<<endl;
        else cout<<"inf"<<endl;
    }
    return 0;
}
