STL：stack用法

2020-05-15 STL常用模板总结 244 字
STL

stack 模板类需要两个模板参数，一个是元素类型，一个容器类型，但只有元素类型是必要的，在不指定容器类型时，默认的容器类型为deque。

一、基本操作

  • 包含头文件#include<stack>
stack<int> s1; // 创建一个栈
s.push(10); // 向栈顶压入一个数为10
s.pop(); // 弹出栈顶元素，无返回值，若栈空则报错
s.top(); // 访问栈顶
s.empty(); // 判断栈是否为空
s.size(); // 返回栈中元素个数

个人实现的简单版本

//栈模板
template <class T>
class stack{
	public:
		bool empty(){return s == 0;}
		int size(){return s;}
		void push(T data){
			s++;
			list* p = new list(data);
			p->next = ptop;
			ptop = p;
		}
		void pop(){
			if(ptop != NULL){
				s--;
				list* p = ptop;
				ptop = ptop->next;
				delete(p);
			}
		}
		T top(){
			if(ptop != NULL){
                return ptop->data;
            }
		}
		stack(){
			s = 0;
			ptop = NULL;
		}
		class list{
			public:
				list(T data): data(data) {next = NULL;}
				T data;
				list* next;
		};
	private:
		int s;
		list* ptop;
};
