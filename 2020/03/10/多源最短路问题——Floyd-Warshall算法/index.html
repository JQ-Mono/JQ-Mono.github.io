多源最短路问题——Floyd-Warshall算法

2020-03-10 经典基础算法 132 字
图论
  • 用于解决多源最短路径问题的算法
  • 时间复杂度为$O(n^3)$
//Floyd算法（关键部分）
//dis[i][j]表示由i到j的路径长度
for(int k = 0; k < n; ++k){
	for(int i = 0; i < n; ++i)
		for(int j = 0; j < n; ++j)
			if(dis[i][j] > dis[i][k] + dis[k][j])
				dis[i][j] = dis[i][k] + dis[k][j];
}
/* 也可以判断是否有负环，但是有些大材小用 */
for(int i = 0; i < n; ++i)
    if(dis[i][i] < 0)
        flag = false; // 表示出现了负环
	}
}
