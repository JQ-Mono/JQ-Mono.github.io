最小生成树——Prim算法

2020-03-13 经典基础算法 314 字
图论
  • 假设一颗只包含一个顶点v的树T。然后贪心地选取T和其他顶点之间相连的最小权值的边，并把它加到T中。处理过程与Dijkstra算法相似
  • 在不优化的情况下，时间复杂度是O(N^2^)，代码写起来也稍微复杂些
  • （真不知道这个算法的意义在哪）
int cost[M][M]; // cost[i][j]表示边(i,j)的权值，不存在的情况下设为INF
int mincost[M]; // 从集合T出发到每个顶点的最小权值
bool used[M]; // 顶点i是否包含在集合T中
int n;
int prim(){
    for(int i = 0; i < n; ++i){
        mincost[i] = INF;
        used[i] = false;
    }
    mincost[0] = 0; // 默认T初始化为只有编号为0这一个顶点
    int ans = 0; // 最小生成树的总权值
    while(true){
        int v = -1; // 挑出在集合T外、T到其权值最小的顶点
        for(int i = 0; i < n; ++i)
            if(!used[i] && (v == -1 || mincost[i] < mincost[v]))
                v = i;
        if(v == -1)
            break; // 所有点都已经被加入T集合
        used[v] = true; // 加入T集合
        ans += mincost[v];
        for(int i = 0; i < n; ++i){
            // 用新加入的点松弛其他所有点
            mincost[i] = min(mincost[i], cost[v][i]); 
        }
    }
    return ans;
}
