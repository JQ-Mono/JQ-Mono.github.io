树状数组（Binary Indexed Tree）

2020-07-02 高级数据结构 816 字
数据结构
  • 求区间[ql,qr]的和，其实只要能求对任意i的区间[1,i]即可，因为[ql,qr]=[1,qr]-[1,ql-1]，也就是说，线段树中右儿子是不必要的，因为有了左儿子和父亲，就能求右儿子的值。
  • 基于这种思路的数据结构就是树状数组。它有着与线段树同样的时间复杂度，但是树状数组的常数更小，代码更短，应用范围也比线段树更加广泛
/* 观察树状数组的构造规律 */
c[1] = a[1];
c[2] = a[1] + a[2];
c[3] = a[3];
c[4] = a[1] + a[2] + a[3] + a[4];
c[5] = a[5];
c[6] = a[5] + a[6];
c[7] = a[7];
c[8] = a[1] + a[2] + a[3] + a[4] + a[5] + a[6] + a[7] + a[8];
// 1-->0001
// 2-->0010
// 3-->0011
// 4-->0100
// 5-->0101
// 6-->0110
// 7-->0111
// 8-->1000
  • 规律：$i$的二进制中从右往左数有$x$个连续的0，那么拥有$2^x$个叶子，为序列A中第$i-2^x+1$到第$i$个元素（共$2^x$个元素）的和。具体求和过程如下：
/* 以求s[7]为例 */
// 7-->0111
c[7] = a[7];
// 减去其二进制最后一个1，得0110
c[6] = a[6] + a[5];
// 减去其二进制最后一个1，得0100
c[4] = a[4] + a[3] + a[2] + a[1];
// 减去其二进制最后一个1，得0000，结果为0，结束
  • 上述求二进制位最低1的操作被称为lowbit，lowbit函数的实现方法有很多：
    • 最常见是这种：x & -x
    • 若不支持访问负数，可以写成：x & (x^(x-1))
    • 每次做的这个操作：x -= x & -x，可以写作x = x & (x-1)
  • 例：$O(logn)$维护单点修改、区间查询
#include<iostream>
#define M 10010
#define lowbit(x) x & -x
using namespace std;
int c[M],n; // c数组保存了一定的a[]序列中的数
int sum(int x){ // 求区间[1,x]的和
    int ans = 0;
    while(x > 0){
        ans += c[x];
        x -= lowbit(x);
    }
    return ans;
}
/* 单点修改的时候复杂度也为O(logn)，只需要修改某些相关联的c[i]即可 */
void add(int x,int value){
    /* 正好求c[i]的祖先只需要让i加上自己的lowbit */
    while(x <= n){
        c[x] += value;
        x += lowbit(x);
    }
}
/* 使用样例：输入区间长度、操作数、每个点的值，每次操作：1 x y表示点x加y，2 x y表示查询区间[x,y] */
int main(){
	int opt, x, y;
	cin>>n>>m;
	for(int i = 1; i <=n ; ++i){
		cin>>x;
		add(i, x);
	}
	while(m--){
		cin>>opt>>x>>y;
		if(opt == 1) 
            add(x,y);
		else if(opt == 2)
            cout<<sum(y) - sum(x - 1)<<endl;
	}
	return 0;
}
  • 变形（树状数组+差分思想）：$O(logn)$维护区间修改、单点查询
#include<iostream>
#define M 500010
using namespace std;
int c[M],n,m;
/* sum函数与add函数不变 */
/* c数组维护的是差分，从而sum(i)求得a[i]的值 */
/* 使用样例 */
int main(){
	int opt, x, y, k, c0 = 0, c1;
	cin>>n>>m;
	for(int i = 1; i <= n; ++i){
        /* 每次读入初始单点后，记录差分，并修改c数组 */
		cin>>c1;
		add(i, c1 - c0);
		c0 = c1;
	}
	while(m--){
		cin>>opt;
		if(opt == 1){
			cin>>x>>y>>k;
			add(x, k);
			add(y + 1, -k);
		}
		else if(opt == 2){
			cin>>k;
			cout<<sum(k)<<endl;
		}
	}
	return 0;
}
